# 基本輸出入元件控制

使用 ATMega328p、LEDx8、按鈕x1

## 電路連接

```
  ┌────────────────────────────────────────────┐
  │  ATMega328P                                │
  │                 ┌────────┐                 │
  │           [RST] │01    28│ A5              │
  │           [RXD] │02    27│ A4              │
  │           [TXD] │03    26│ A3    (LED7)    │
  │               2 │04    25│ A2    (LED6)    │
  │              ~3 │05    24│ A1    (LED5)    │
  │               4 │06    23│ A0    (LED4)    │
  │           [VCC] │07    22│ [GND]           │
  │           [GND] │08    21│ [AREF]          │
  │          [XTAL] │09    20│ [VCC]           │
  │          [XTAL] │10    19│ 13    (LED3)    │
  │              ~5 │11    18│ 12    (LED2)    │
  │              ~6 │12    17│ 11~   (LED1)    │
  │    (BTN0)     7 │13    16│ 10~   (LED0)    │
  │               8 │14    15│ 9~              │
  │                 └────────┘                 │
  └────────────────────────────────────────────┘
```

## 練習一

LED 初始狀態為全滅，當按鈕按ㄧ下，八顆 LED 全亮全滅三次。

## 練習二

LED 初始狀態為右邊四顆亮，當按鈕按一下，左邊四顆 LED 全亮，再按一下，右邊四顆 LED 全亮，依此循環。

## 練習三

LED 初始狀態為最右邊一顆亮，當按鈕還未被按下，LED 向左旋轉且循環，當按鈕按下，LED 改為向右旋轉且循環。
